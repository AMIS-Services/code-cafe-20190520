## Getting started

### Docker

```
$ docker build -t recharts:dev .
$ docker run -v ${PWD}:/app -v /app/node_modules -p 3001:3000 --rm recharts:dev
```

### Local dev

```
npm install
npm start
```

of

```
yarn
yarn start
```

### Making changes

Go ahead and make some changes in to the files in `src/`, your browser will hot reload and show your changes.
